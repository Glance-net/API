Glance API Homepage

Integrate Glance into your customer service workflows with secure, flexible APIs.

API Description

Portal

Provides a comprehensive set of endpoints for administrators to configure and manage group-level settings within the Glance Portal.

Roles

Enables administrators to manage role-based access control by providing endpoints to create, retrieve, update, and delete user roles within a group.

Users

Enables administrators to manage user accounts within a group by providing endpoints to create, retrieve, update, and delete users.
